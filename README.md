# MLB予測サイト

NPB版と同じ方法論をMLBメジャーリーグに適用した勝敗予測・分析ツール。

## ファイル構成

- `index.html` — 当日の予測ページ（毎日更新）
- `record.md` — 予測記録・累積成績・Brierスコア
- `analysis.md` — 試合ごとの振り返り分析ログ
- `CLAUDE.md` — Claude Code 運用ガイド

## 予測ロジック概要

1. シーズン勝率差（Log5法で基本勝率を算出）
2. 先発投手ERA・WHIP・K/BB差（両先発を個別評価）
3. ホームアドバンテージ +4pt（MLB統計的長期平均）
4. 上限65%・ERA5.00以上先発は55%以下に抑制

## グレード定義

| グレード | 確率帯 | 推奨 |
|---|---|---|
| A | 70%以上 | 厳選ピック |
| B | 58〜69% | 標準予測 |
| C | 50〜57% | 見送り推奨 |
