## 検証ルール

- 「予想勝者」が勝利した場合 → 的中(○)
- 「予想勝者」が敗戦または引き分けの場合 → 不的中(×)
- 雨天中止・ノーゲームは検証対象外（集計テーブルは「中止」）
- 延長戦での結果も通常通り検証対象
- 累積的中率 = 的中数 ÷ 検証済み試合数

## Brierスコア算出メモ

```
Brier = Σ(p - o)² / N
  p = 予測確率（0〜1）、o = 実結果（的中=1 / 不的中=0）
  0に近いほど優秀（完全予測=0、ランダム=0.25）
```

```
= 0.9681 / 4 = 0.242
```
