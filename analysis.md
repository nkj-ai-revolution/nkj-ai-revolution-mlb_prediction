## 2026-07-04 振り返り（7/5夜版で全15試合確定・9勝6敗 60.0%）

> ⚠️ 注記: 本更新時点（2026-07-05）でMLB公式スコアボード/ESPNを直接確認したところ、2026-07-05（本日）の15試合は全試合「開始前」であり、本日分の的中判定はまだ行えない。そのため今回の振り返りは、前日7/4に行われ本更新でようやく確定した残り11試合を対象とする。7/5当日の振り返りは試合終了後の次回更新（夜版）で追記する。

### 的中した試合（なぜ当たったか）

- **NYM@ATL（ATL 65%予測 → ATL 14-3で的中）**: チーム力差（ATL.588 vs NYM.414）と先発セールの安定感を根拠にしたロジック通り。セール自身は5回3失点とやや荒れたが、救援リッチーが3回無失点で締め、打線が14得点と根拠通り優位を発揮。想定スコア(6-3)より大差になったが、勝敗という点では根拠が正しく機能。
- **PHI@KC（PHI 60%予測 → KC1-6PHIで的中）**: 先発FIP差（ルザード3.15 vs ワチャ3.92）が的中。ルザードが6回1失点9奪三振と支配的に投げ、根拠通りの決着。
- **CIN@BAL（BAL 54%予測 → CIN5-8BALで的中）**: 「H.グリーンはシーズン初登板でMLB成績未取得・保守評価（グレード上限B・−2pt）」という事前の警戒フラグが的中。グリーンは3.1回8失点と大炎上し、データ欠損時の保守運用ルールの妥当性を裏付けた。
- **SF@COL（SF 55%予測 → COL4-6SFで的中）**: レイの好調継続（6回3失点）とCOLブルペン最下位級という評価が的中。COL先発サリバンも5.2回6失点で崩れ、先発・救援両面でSF優位が反映された。
- **BOS@LAA（BOS 55%予測 → LAA1-8BOSで的中）**: アルデジェリのFIP4.8+キャップ発動が的中。グレイが6回1失点と好投し、キャップ運用の正しさを再確認。
- **SD@LAD（LAD 55%予測 → LAD3-0SDで的中）**: キャニングのERA5+/FIP4.8+キャップ発動通り。山本由伸が7回無失点10奪三振と、55%キャップがむしろ実力差（本来68.8%相当）に対して保守的すぎた可能性がある一戦。
- **MIA@ATH（MIA 54%予測 → ATH2-7MIAで的中）**: アルカンタラが8回1失点8奪三振、シバレー(Civale)は4.2回4失点。先発FIP差(1.32)通りの決着。

### 外れた試合（何が予測と違ったか）

- **CWS@CLE（CLE 60%予測 → CWS3-1CLEで外れ）**: 先発レイヤーは的中側だった（メシック5回1失点は根拠通りの好投）。外れた原因は**ブルペン恒常レイヤー**——「CLEブルペンは絶好調・フレッシュ」と評価していたが、実際は自軍の救援ヘリンが被弾し敗戦投手となった。CWS側の消耗（延長10回サヨナラ負けの疲労）という前提は誤りではなかったが、CLE自身の救援陣の日替わりの不安定さまでは織り込めなかった。ブルペンの「シーズン通算成績が良い」ことと「当日の一発の出来」は別物であることを示す一例。
- **TB@HOU（TB 57%予測 → HOU10-8TBで外れ）**: 先発FIP差(0.40<0.5)により先発レイヤーを無効化していたが、両先発（ブラウン4回7失点・ラスマッセン5回5失点）がともに崩れる乱打戦となり、**先発レイヤーではなく打線レイヤー**（アストロズ打線の得点力）が決定打になった。FIP差僅少ルールは妥当だったが、両先発の炎上リスクまでは事前に織り込めなかった。
- **STL@CHC（CHC 53%予測 → CHC0-3STLで外れ）**: 今永(4.2回2失点)は許容範囲の投球内容だったが、**打線レイヤー**でCHC打線が完封負け。事前に「カブス投手陣が深刻な人員不足（12投手IL在籍）」と保守設定していたのは救援面の懸念だったが、実際は打線が沈黙したことが敗因で、想定していたレイヤーとズレがあった。
- **AZ@MIL（MIL 55%予測 → AZ4-3MILで外れ）**: 事前に「ウッドラフは肩の炎症からの復帰間もなく41.2回・球数管理下の小サンプルで信頼度は割り引く」と明記していた懸念が的中する形で外れた。ウッドラフは3.2回3失点で早期降板し、対するケリー（ERA5.84でキャップ対象だった劣勢評価の先発）が5回2失点で先発逆転。**先発レイヤー**で「小サンプル・球数管理」のリスクフラグを立てていたにもかかわらず、確率計算には十分反映しきれていなかった可能性がある。

### 今日の傾向

- ブルペンの「シーズン通算FIP/ERAが良い」ことは当日の安定登板を保証しない（CWS@CLEのヘリン被弾が好例）。恒常レイヤーは方向性の参考にはなるが、単発試合では逆転が起こりうる点を再認識。
- FIP差僅少（<0.5）ルールで先発レイヤーを無効化した試合（TB@HOU）で両先発が崩れる乱打戦になった。このルール自体は「先発で差がつかない」ことの予測としては機能しており問題はない。
- 「小サンプル・球数管理」フラグを付けた先発（ウッドラフ）が実際に早期降板したのは、注記自体の妥当性を裏付ける結果。今後もこの種のフラグは確率への反映（保守化）をやや強めることを検討する材料になるが、1試合の結果でロジックを恒久的に修正するには材料不足のため、通常運用を継続する。
- 7/4は9勝6敗(60.0%)で着地し、累積的中率は55.8%→56.4%に改善、Brierスコアは0.247で横ばい。

### 翌朝への申し送り

- 2026-07-05（本日）の15試合は本更新時点（MLB公式スコアボード確認）で全試合開始前。結果確定後の夜版更新で的中判定・累積再計算・振り返りを行う。
- CLE・CHCなど、救援陣の「シーズン通算の良さ」だけでなく直近の登板負荷（連投・被弾傾向）を毎回個別に確認する運用を継続する。
- ウッドラフ（MIL）やビーバー（TOR）など「故障明け・球数管理下」の先発は今後も登板イニング数に上限がかかりやすく、好投していても5回前後で降板するリスクを織り込んで評価する。
- BAL・SFなど、故障者リスト入り（クローザー等）が判明した相手チームに対する優位性評価は、実際の結果でも継続して支持されており、IL異動の即時反映運用は妥当と判断。

## 2026-07-05 振り返り（7/6朝版時点・13/15試合確定・5勝8敗 38.5%）

> ⚠️ 注記: 本更新時点でSD@LAD・BOS@LAAの2試合は結果を確認できず、次回更新で振り返りに追加する。

### 的中した試合（なぜ当たったか）

- **STL@CHC（CHC 53%予測 → CHC 6-4勝で的中）**: 僅差評価が的中側に転んだケース。カブス投手陣の人員不足を織り込みつつも本拠地の底力が発揮された。
- **SF@COL（COL 54%予測 → COL 7-6勝で的中）**: 両先発ともERA5+/FIP4.8+キャップ該当の「荒れ球カード」評価通り、乱打戦の末にホームのCOLが競り勝った。
- **ARI@MIL（MIL 55%→最終版53%予測 → MIL 3-2勝で的中）**: 前日のウッドラフ早期降板によるブルペン消耗を織り込んだ保守化判断が的中側に。
- **ATH@MIA（MIA 51%→最終版52%予測 → MIA 9-8勝で的中）**: ATH先発の早期降板による救援消耗差を反映した上方修正が的中。
- **TOR@SEA（SEA 56%予測 → SEA 4-0勝で的中）**: 前日の大勝でブルペンを温存できていたSEAが順当に完封勝利。

### 外れた試合（何が予測と違ったか）

- **NYM@ATL（ATL 63%予測 → NYM 10-9勝で外れ、本日最大の損失）**: チーム力（ATL.591 vs NYM.411）と先発差を根拠にした高確信度の本命だったが、9回にATLが猛追するも守護神D.ウィリアムズがセーブを守り切りNYMが逃げ切った。ホーム高確率予測（60%超級）がまたも裏目に出るパターンで、Grade Bクラスの「大差予想」が一発の接戦・逆転劇に弱いという構造的な課題を改めて示した。
- **PIT@WSH（WSH 58%予測 → PIT 11-5勝で外れ）**: 先発差（カバリERA3.69 vs チャンドラーERA4.62）は妥当な評価だったが、実際は打線・ブルペン全体でPITが上回り大差がついた。**打線レイヤー**でのズレ。
- **BAL@CIN（BAL 54%予測 → CIN 3-2勝で外れ）**: 先発差でBAL優位と評価していたが、僅差の投手戦でCINが競り勝った。
- **MIN@NYY（NYY 56%予測 → MIN 6-1勝で外れ）**: 先発FIP逆転（ライアン(MIN)がウェザーズ(NYY)を上回る）を織り込んで抑制した56%だったが、それでも不十分な評価で大差負け。NYYホーム高確率設定の課題が継続。
- **KC@PHI（PHI 55%予測 → KC 5-2勝で外れ）**: 両先発ともERA5+/FIP4.8+キャップ該当の「荒れ球カード」だったが、KC側がクローザー交代（A.ランジ）で好投し逃げ切った。
- **TB@HOU（TB 56%予測 → HOU 2-0勝で外れ）**: 前日の消耗を反映し保守化したTB予測だったが、HOU先発ランバートが完封級の内容で完全に上回った。
- **DET@TEX（TEX 53%予測 → DET 6-3勝で外れ）**: マイズ(DET)のFIP優位を織り込んで抑制していた53%だったが、それでも実際の実力差はさらに大きかった。
- **CWS@CLE（CLE 55%予測[最終版で58%B→55%Cに格下げ済み] → CWS 7-6勝で外れ）**: 予告先発変更（フェデ→マーフィー）を反映しグレード格下げした判断は妥当だったが、それでも僅差でCWSが競り勝った。

### 今日の傾向

- **13試合中5勝8敗(38.5%)は今季ワースト級の一日。** Grade B 2試合（ATL 63%・WSH 58%）が両方とも外れたのが大きく響いた。ただし免責事項通り「63%の本命でも約37%は負ける」範囲内の分散であり、単発の悪い一日でロジックを過剰修正すべきではない。
- ホーム予測の的中率が今回も低調（8戦中3勝、対してビジター予測は5戦中2勝）で、累積のホーム/ビジター乖離（50.6% vs 58.8%）をさらに拡大させた。ホーム側の保守キャップ（NYY等）は妥当な運用として継続する。
- NYM@ATLのように、ホーム高確率帯（60%超）の予測が土壇場の逆転劇で外れるケースが今季複数回発生している。守護神・終盤リリーフの一発のリスクは、シーズン通算のブルペン成績だけでは捕捉しきれない。

### 翌朝への申し送り

- SD@LAD・BOS@LAAの2試合は結果未確認のため、次回更新で確定反映・振り返りを行う。
- ホーム高確率帯（58%以上）の予測が接戦で終盤に逆転されるケースが繰り返し発生しており（7/4 CWS@CLE、7/5 NYM@ATL等）、終盤の守護神・セットアッパーの調子は「シーズン通算成績」だけでなく直近の被弾傾向を可能な限り確認する運用を継続する。
- 7/6はブルペンFIP/ERAのチーム集計がFanGraphs等から全チーム分取得できなかった（データソース側の一時的な問題の可能性）。翌日以降も同様の欠損が続く場合は代替ソース（Baseball Reference等）での再取得を試みる。
- 単発の悪い一日（5勝8敗）でロジックを恒久的に修正するには材料不足。数十試合単位での評価を継続し、通常運用を維持する。
