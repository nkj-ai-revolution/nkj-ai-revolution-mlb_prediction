## 2026-07-18 部分確定振り返り（7/19朝版〖516試合中11試合確定・4勝7敗 36.4%）

- **MIN@CHC（CHC 60%予測B → CHC 6-2勝で的中）**: ボイド(CHC)が試合を作り、CHCブルペン優位の評価通りの決着。
- **MIA@MIL（MIL 57%予測C → MIL 8-6勝で的中）**: FIP差僅少(0.06)で先発レイヤーを無視した評価だったが、MILブルペン(メギル)がMIA主力リリーバー2名IL入りという劣勢を突いて逃げ切った。

### 外れた試合（何が予測と違ったか）

- **TEX@ATL（ATL 59%予測B → TEX 7-6勝で外れ）**: 先発FIP差・ホームアドバンテージを根拠にATL優位としていたが、レンジャーズが6回に4点を奧い逆転。**打線レイヤー**でTEXが上回った。
- **TB@BOS（TB 57%予測C → BOS 7-6勝で外れ）**: 先発片方(サンドバル)がTJ手術明けの小サンプルでデータ欠損評価だったが、接戦の末にBOSが僅差を制した。
- **CWS@TOR（CWS 55%予測C → TOR 1-0勝で外れ）**: ビーバー(TOR)のTJ復帰後不振(ERA9.00超)をキャップ対象としていたが、ブルージェイズ投手陣がCWS打線を完封。**先発・救援レイヤー**双方でTOR優位という見誤り。
- **SD@KC（SD 53%予測C → KC 6-1勝で外れ）**: キャニング(SD)のERA6.47+キャップ該当通り崩れたが、SDの好調ブルペンが登板機会を得る前に序盤で大量失点し試合を決められた。
- **BAL@HOU（HOU 53%予測B → BAL 4-2勝[延長11回]で外れ）**: 両先発互角評価だったが延長11回までもつれ、**救援レイヤー**でBALが上回った。
- **PIT@CLE第1試合（CLE 55%予測C → PIT 7-1勝で外れ）**: データ欠損で保守評価していたPIT先発ジョーンズが試合を作り、PITのチームwRC+111という打線優位が素直に反映された。**先発・打線レイヤー**でPIT優位という見誤り。
- **WSH@ATH（ATH 51%予測C → WSH 23-4勝で外れ、本日最大の損失）**: 「WSHブルペンがリーグ最悪級」という弱点評価を根拠にATH優位としていたが、実際は**打線レイヤー**でWSH打線が23得点と大爆発。7/17のWSH@ATH（当時はATH 54%予測で同様に外れ）と全く同じパターンの再現であり、「相手ブルペンが弱い＝自チーム優位」の単純化ロジックが中2日で再び同一カードで裏目に出た。

### 今日の傾向

- WSH@ATHが7/17・7/18と連続して「相手(WSH)ブルペンの弱さ」を根拠にATH優位とする評価が外れ、いずれもWSH打線の大量得点(23-4・23-4級の連続大勝)で決着した。同一カードで同じ失敗パターンが連続再現したことは、「ブルペンが弱い相手＝自チーム優位」という単純化ロジックの妃当性を中期的に再検証すべき根拠として重みを増している（ただ、2試合はまだ少数サンプルであり、通常運用は継続する）。
- PIT@CLEでは、先発情報が欠損していた側(PIT)がむしろ試合を作り、打線 wRC+の優位(111 vs 92)が素直に反映される決着になった。データ欠損時の保守評価が常に「弱い」方向に外れるとは限らない一例。
- 11試合中4勝7敗(36.4%)は7/17(35.7%)に続き2日連続の低調な的中率。ただし免責事項通り単一試合・単日の結果でロジックを過剰修正すべきではなく、通常運用を継続する。

### 翌朝への申し送り

- SF@SEA・DET@LAA・PIT@CLE第2試合・STL@AZの4試合は次回更新で確定反映・振り返りを行う。
- WSH@ATHで2試合連続再現した「相手ブルペンの弱さ＝自チーム優位」ロジックの単純化リスクは、15〜20試合単位の中期検証課題として引き続き蓄積する。打線側の得点力(wRC+)が高いチームに対しては、相手ブルペンの弱さだけで大幅な確率補正をしないよう留意する。
- ドジャース@ヤンキースは7/18開催予定分が雨天順延となり、本日7/19にダブルヘッダー(第1試合12:35 PM ET・第2試合7:20 PM ET)として振替開催。第1試合は山本由伸(LAD)が予告先発、第2試合は本更新時点で両先発とも未定。
- 本日(7/19)は多くの試合でFIP/xFIPがFanGraphs等から直接取得できず、ERA/WHIPによるフォールバック評価やデータソース間の数値不一致(今永・ホームズ等)が例年以上に多い一日だった。データ取得ソースの安定性は中期的な課題として意識する。

## 2026-07-21 朝版 データ取得における留意事項（振り返り対象なし・7/20結果は未確定）

- MIN@CLE（MIN先発）・BAL@BOS（バズ/ブラディッシュ）・ATH@AZ（パーキンス/スプリングス）の3試合は先発投手の特定が試合直前まで確定できておらず、最終分析版で必ず再確認する。
